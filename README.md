# Better Pause (Geode Mod)
Разработчик: Steam4ch

Это сгенерированный шаблон мода для Geometry Dash с помощью Geode.
Мод содержит пример кода для модификации экрана паузы (Better Pause).

## Как редактировать:
Код находится в `src/main.cpp`. Откройте его в вашем редакторе (например, VS Code).

## Как компилировать:
### Способ 1: Автоматически (рекомендуемо новичкам)
1. Создайте новый публичный репозиторий на [GitHub](https://github.com/).
2. Загрузите туда все файлы из этой папки.
3. Перейдите во вкладку **Actions**.
4. GitHub автоматически скомпилирует файлы `.geode` для Android, Windows и Mac!
5. Скачайте их из "Artifacts" в самом низу результата сборки (Build).

### Способ 2: Вручную
Если у вас установлен Geode CLI и нужные C++ компиляторы (MSVC, Clang):
1. Откройте консоль в этой папке.
2. Введите `geode build`.
